O README é um arquivo que pode conter informações escolhidas pelo criador do programa, que é mostrado para o usuário.

Para se trabalhar com versionamento utilizamos as seguintes ferramentas:

  1. Git: para repositório local
  2. Github: para repositório remoto
  3. Visual Studio Code: para criar, editar e controlar códigos
